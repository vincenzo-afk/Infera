# 04 — User Stories

### Chaos Mode Toggle
As a user, I want to turn voice processing on and off with one tap, so that I can quickly switch between my normal voice and the effect during a call or recording.

### Preset Selection
As a user, I want to pick from ready-made voice presets, so that I can get a good effect instantly without manually tuning anything.

### Manual Effect Tuning
As a power user, I want to adjust individual effect parameters, so that I can craft my own unique voice sound.

### Volume Boost
As a user, I want to boost the output loudness up to 500%, so that the effect is impossible to miss or ignore during a call.

### Custom Preset Save/Load
As a user, I want to save my own tuned settings as a preset, so that I don't have to re-tune everything every time I use the app.

### Foreground Service & Notification
As a user, I want a clear notification while the effect is active, so that I always know my voice is currently being processed.

### Audio Focus Handling
As a user, I want the app to behave correctly if I get a phone call while using it, so that nothing crashes or conflicts.

### Permissions
As a user, I want to understand why the app needs microphone and notification access, so that I feel confident granting them.

### Background Stability
As a user, I want the app to keep running if I switch to another app briefly, so that the effect doesn't cut out mid-call.

### Preview Before Activation
As a user, I want to preview an effect before turning it on live, so that I can choose the right one without surprising anyone on a call.
